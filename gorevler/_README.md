# Anomali Test Görevleri

Bu klasör, ekip üyelerinin kendi anomalilerini test etmeleri için oluşturulmuş görev dökümanlarını içerir.

## Nasıl Kullanılır?
1. Kendi adınızın geçtiği dosyayı bulun (Örn: `Ahmet_Bsg_ahmet-ddos.md`).
2. Dosyadaki adımları takip ederek:
   - UI üzerinden (User Mode) testi gerçekleştirin.
   - Developer Mode üzerinden JSON planı veya **Python SDK** ile doğrulama yapın.
3. Aldığınız sonuçların ekran görüntüsünü ve raporunu dosyanın sonuna ekleyin (veya ayrı bir rapor dosyasına).

## Python SDK Kullanımı
- `sdk/` klasöründeki `evcs_attack.py` dosyasını kullanın.
- `attack_example.py` dosyası örnek saldırı kodlarını içerir.
- Saldırı yapabilmek için **/developer** sayfasından belirlediğiniz `DEV_API_KEY`'i kullanmanız gerekir.
